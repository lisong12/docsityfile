# 菜单列表#

### 功能介绍 ### 

菜单列表为vue + element UI搭建导航菜单，具有可折叠式、动态加载效果，加载数据从后台获取并渲染。
   
  

### 使用方法 ###
        <sf-menu :menu="menu" @setUrl="setUrl"></sf-menu>
